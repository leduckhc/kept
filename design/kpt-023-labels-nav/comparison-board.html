Direction B
Icon Pill Switcher
A compact pill of 4 icon buttons lives in the top-right corner of the toolbar. Active view is the only element highlighted. View name appears as a small label chip below the toolbar.
Discoverability Medium — icons visible but no labels by default
Screen real estate Minimal — uses existing toolbar row
Tap target 28px each — small, but familiar pattern

Direction C
Title-as-Navigator
The toolbar title IS the navigation control. "Inbox ▾" is tappable — it slides down a compact tray with all 4 views as icon + label cells. Zero extra chrome; zero permanent real estate cost.
Discoverability Lower — requires knowing the chevron is interactive
Screen real estate Zero cost when tray closed
Tap target Full toolbar title width — easy to hit

Denisa's recommendation
Direction C — Title-as-Navigator
  • Zero chrome cost: the toolbar title becomes the navigator — inbox real estate is untouched, which is what Kept's ultra-minimal taste demands.
  • Discoverable enough at alpha stage: "Inbox ▾" chevron is a legible affordance for the small engaged user base; can add onboarding tooltip once in beta.
  • Tray interaction scales: if Kept adds Labels or Filters later, the same tray expands naturally — whereas the tab bar would overflow at 5+ items.
