Summary
Option 1 (9+ cap): Simplest. Fixed badge size. Loses precision above 9.
Option 2 (thick ring): No numbers at all — weight = importance. Subtle, ambient.
Option 3 (boolean dot): Cleanest. Dot = "is group", count in text. Never scales.
Option 4 (ring gap): Count lives inside the ring's gap. Elegant, contained, caps at 99+.

My pick: Option 4 (ring gap) — it treats the count as part of the ring's design language rather than slapping a badge on top. The gap is intentional, the number fills it. Scales to 99+ cleanly. Individual threads = closed ring (no gap) = obvious differentiation.
