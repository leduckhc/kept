# Disable "Sent with Spark" tagline

**Source:** https://sparkmailapp.com/help/sending-emails/disable-sent-with-spark-tagline

---

To disable the "Sent with Spark" tagline, please follow these steps:

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/general/how-to-find-spark-settings).
2. Select **Composer > Signatures**.
3. **Display "Sent with Spark" tagline** > Disable the toggle.

2. Select **Composer > Signatures**.
3. **Display "Sent with Spark" tagline** > Disable the toggle.
