# Set Spark as a default email client

Source: https://sparkmailapp.com/help/general/set-spark-as-a-default-email-client
Scraped: 2026-05-31T10:49:56Z

---

[Mac](#) [Windows](#) [Android](#) [iOS](#)

If you use the App Store version:

1. Make sure that Spark Desktop is installed.
2. Open the Mail app.
3. Go to **Mail** **>** **Settings** at the top, then click **General**.
4. Choose **Spark Desktop**from the Default email reader menu.

If you use the website version:

1. Make sure that Spark Desktop is installed.
2. Open [**Spark Settings**](https://sparkmailapp.com/help/general/how-to-find-spark-settings) **>** **General**.
3. Make Spark the default email client > Make default.

Now, your device will open Spark Desktop when you click an email link or want to email something from other apps.

If you are using Spark on Windows, you cannot currently set Spark as your default email client.

If you are using Spark on Android, you cannot set Spark as your default email client. At the moment, you can only select the app that opens mailto:// links by default:

1. Open Browser app > Open the page that contains an email address.
2. Tap on this address > **Open with.**
3. **Spark** **>** **Always**.

1. Make sure your device runs at least iOS 16.
2. Open the **Settings** app.
3. Scroll down and select **Spark**.
4. Tap **Default Mail App** and tick **Spark**.

If you use iOS 18.2 and later:

1. Open the **Settings** app.
2. Scroll down and select **Apps**.
3. **Default Apps** **>** **Email.**
4. Select **Spark**.

Now, your device will open Spark when you tap an email link or want to email something from other apps.
