# Manage emails with swipes

**Source:** https://sparkmailapp.com/help/manage-your-inbox/manage-emails-with-swipes

---

You can easily process your Inbox with swipes. With a simple swipe, you can effortlessly glide between emails and email groups to enhance productivity.

[Your browser does not support HTML5 video.](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1746454770049-cuatomize%20swipes.mp4)

**Note:**Swipe preferences don't sync across all your devices.

## How to configure swipes

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/help/general/how-to-find-spark-settings).
2. Enter the Swipes section.
3. Select an action for each swipe.

Swipes are currently not available in Spark Desktop on Windows

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588252227326-1588252227326.png)  at the top left.
2. Tap **Settings** ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588252235295-1588252235295.png)
3. Open **Personalization > Swipes**.
4. Tap a swipe.
5. Select one of the actions.

**Note:**Spark suggests to use the options you set as swipe actions (e.g. pin or delete) as batch actions when you select multiple emails. Learn [how to use batch actions](https://sparkmailapp.com/manage-your-inbox/perform-actions-on-multiple-emails).
