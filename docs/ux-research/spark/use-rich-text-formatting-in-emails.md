# Use rich text formatting in emails

**Source:** https://sparkmailapp.com/help/sending-emails/use-rich-text-formatting-in-emails

---

Spark offers a variety of rich text formatting options. You can select the text color and text highlight color, create lists and indents, add quotes, headings, and hyperlinks.

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. In the email composer, type or paste your message.
2. Select the text you want to format and choose one of the options on the toolbar.

![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1744198600441-Screenshot%202025-04-09%20at%201.24.31%E2%80%AFPM.png)

You can also use the keyboard shortcuts: **Cmd B** for **bold**, **Cmd I** for *italic*, and **Cmd U** for underline.

When you start a row with a dash, Spark automatically starts an unordered list.

1. In the email composer, tap the **Text Style** ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1744198825581-Screenshot_20250409_133817_Spark.jpg)icon.
2. Select the text and choose one of the options on the toolbar. Scroll the toolbar to the right to reveal more actions.
3. After formatting text, tap the checkmark icon on the toolbar.  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1744198559158-Screenshot_20250409_133309_Spark.jpg)

If you wish to remove formatting, select the text, scroll the toolbar to the right, and tap the **Remove Style** icon.
