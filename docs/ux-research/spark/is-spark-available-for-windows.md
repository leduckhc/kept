# Is Spark Available for Windows?

**Source:** https://sparkmailapp.com/help/general/is-spark-available-for-windows

---

Spark is a cross-platform email client available on macOS, Windows, iOS, and Android platforms. You can download Spark for Windows [here](https://downloads.sparkmailapp.com/Spark3/win/dist/3.0.28.37418/Spark.exe).

[Click here](https://apps.microsoft.com/detail/xpfcs9qjbkthvz?hl=en-US&gl=US) to download the Microsoft Store version.

**Note:**Spark on Windows requires Windows version 10 or later.
