# Create a Smart Folder on iOS

Source: https://sparkmailapp.com/help/manage-your-inbox/create-a-smart-folder-on-ios
Scraped: 2026-05-31T10:51:01Z

---

In Spark on iOS, you can set up a smart folder to filter emails by sender, attachments, or content. A smart folder gives you quick access to chosen emails and makes your inbox more organized. For example, if you travel frequently, set up a smart folder for your tickets and reservations, and they appear in a separate folder on the sidebar.

Spark understands natural language; you can type “bookings from Airbnb this year,” and the app finds the referenced emails to create a folder.

Please check [this article](https://sparkmailapp.com/help/manage-your-inbox/create-smart-folder) to learn more about creating Smart Folders on Desktop devices.

**Tip:**For Gmail accounts, Smart Folders show emails from all folders except Trash and Spam. For non-Gmail accounts, Spark looks for emails in Inbox and Archive unless you specify the folder name in your search query.

## Set up a smart folder

Please note: Smart folders exist only in Spark and don't display on the web page of your email account.

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588250862214-1588250862214.png)  at the top left.

2. On the sidebar, select **Edit list.**  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1773834102259-Screenshot%202026-03-18%20at%2012.38.55%E2%80%AFPM.png)

3. Scroll down and tap **Add Smart Folder**.  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1773834198093-Screenshot%202026-03-18%20at%2012.36.41%E2%80%AFPM.png)

4. Tap **Please, select filter** and start searching for the needed emails. For example, if you want all the emails with PDF attachments from your colleague, enter “pdf from…”, start typing a person’s name or email, and Spark suggests the results. Learn more on [how to use search in Spark](https://sparkmailapp.com/manage-your-inbox/use-smart-search).

**Note:**If you search for emails from multiple people, choose the first person on the list, insert a comma, and start typing the second name or email.

5. After creating the filter, tap **Save**.
6. In the **Name** field, type the name of your folder.
7. In the **Accounts** section, select the accounts to use with this smart folder.
8. Tap **A****dd**at the top right.
9. To add your new smart folder to the sidebar, tap the plus sign near it.  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1773835003222-Screenshot%202026-03-18%20at%2012.53.06%E2%80%AFPM.png)

10. Tap **Save** at the top right. Now, this folder is available on the sidebar.

## Edit a smart folder

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588250903620-1588250903620.png)  at the top left.
2. On the sidebar, tap **Edit list.**
3. To edit a smart folder, you need to remove it from the sidebar first. Find your folder on the list and tap the minus sign near it.  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1773835126043-Screenshot%202026-03-18%20at%2012.53.50%E2%80%AFPM.png)

4. Scroll down and tap the **More** icon ![](https://d3pbdh1dmixop.cloudfront.net/spark/help/Icons/iOS/smiGeneralMore.png) near the smart folder.  
   ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1773835178368-Screenshot%202026-03-18%20at%2012.54.23%E2%80%AFPM.png)

5. Select **Edit**.
6. Here, you can change the filter rules, folder name, and manage the accounts for your smart folder.
7. After applying changes, tap **Save**. The edited folder is added to the sidebar.

## Delete a smart folder

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588250936199-1588250936199.png)  at the top left.
2. On the sidebar, tap **Edit list**.
3. To edit a smart folder, you need to remove it from the sidebar first. Find your folder on the list and tap the minus sign near it.
4. Scroll down and tap the **More** icon ![](https://d3pbdh1dmixop.cloudfront.net/spark/help/Icons/iOS/smiGeneralMore.png) near the smart folder.
5. Select **Remove Folder**.

**Note:**When you delete a smart folder, the emails it contains aren’t affected. They are still available in your inbox.

Please make sure you delete **a smart folder, not an ordinary folder**(e.g. Important, Drafts, etc.). The emails in ordinary folders will be deleted, and this action can’t be undone. You can only delete **a smart folder**without losing your emails.
