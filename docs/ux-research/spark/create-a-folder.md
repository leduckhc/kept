# Create a Folder

Source: https://sparkmailapp.com/help/manage-your-inbox/create-a-folder
Scraped: 2026-05-31T10:50:52Z

---

In Spark, you can easily create folders and subfolders to organize your emails. You can also manage folders on the native web page of your email account (Gmail, iCloud, etc.), and Spark syncs the changes across all your devices.

## Add a new folder

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. Open Spark Desktop and press **⌘ Shift U**.
2. Type the folder's name and click an account where to create the folder.

You can also create a folder from the sidebar:

1. If your sidebar is [Locked](https://sparkmailapp.com/help/tips-tricks/new-expanded-sidebar), open Spark Desktop and click ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1661935795313-Sidebar.png)on the sidebar.
2. Click **More ...**and click**+**next to the account you want to create a folder for.

## Create a subfolder

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. If your sidebar is [Locked](https://sparkmailapp.com/help/tips-tricks/new-expanded-sidebar), open Spark Desktop and click ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1661935795313-Sidebar.png) icon on the sidebar.
2. Click **More ...**and click**+**next to the folder you want to create a subfolder for.

1. If your sidebar is [Locked](https://sparkmailapp.com/help/tips-tricks/new-expanded-sidebar), open Spark Desktop and click ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1661935795313-Sidebar.png) icon on the sidebar.
2. Click **More ...**and click**+**next to the folder you want to create a subfolder for.

1. Open the sidebar and tap **Edit list**.
2. Scroll down and tap the **More** icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588254297557-1588254297557.png)  near the folder.
3. Select **Create Subfolder**.
4. Type the folder name.
5. Select **Add**at the top right.
6. Tap the plus icon near your new folder to keep it on the sidebar.
7. Select **Save** at the top right of the sidebar.

If you pinned a folder to the sidebar and want to create a subfolder inside it, you need to unpin it first. Find the needed folder on the list and tap the minus sign near it. Then, scroll down, tap **More**![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588254305153-1588254305153.png) near the folder and select **Create Subfolder**.

## Move emails to a folder

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. Select the needed emails in your Inbox and press **⌘ Shift M**.
2. Select a folder to move this email.

1. Select the needed emails in your Inbox and press **CTRL Shift M**.
2. Select a folder to move this email.

1. Open a needed email.
2. Tap the **More**icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588254297557-1588254297557.png) at the bottom.
3. Select **Move**.
4. Choose a folder to move this email. To switch between your accounts, tap your email address under**Move to** at the top.

To move multiple emails at once, long tap one of the emails in your inbox and then tap the other messages you want to select. ## Delete a folder

**Note:**When you delete a folder, all emails it contains are erased permanently. You can’t undo this action.

[Mac](#) [Windows](#) [Android](#) [iOS](#)

To delete a folder, right-click it on the sidebar and select **Delete.**

To delete a folder, right-click it on the sidebar and select **Delete.**

1. Open the sidebar and select **Edit list**.
2. Scroll down and tap the **More** icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588254297557-1588254297557.png)  near the folder.
3. Select **Delete Folder**.
4. In a pop-up message, tap **Delete folder.**

In case you pinned a folder to the sidebar, you need to unpin it first. Find your folder on the list and tap the minus sign near it. Then, scroll down, tap **More**![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588254297557-1588254297557.png) near the folder and select **Delete Folder**.
