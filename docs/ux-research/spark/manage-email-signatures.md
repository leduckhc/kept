# Manage email signatures

**Source:** https://sparkmailapp.com/help/sending-emails/manage-email-signatures

---

Spark allows you to create a pre-arranged signature which appears automatically at the bottom of the email composer. You can have multiple signatures and set the default ones for different accounts to save time while writing emails.

## Create a signature

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/general/how-to-find-spark-settings).
2. Select **Composer >****Signatures**.
3. Click the **Create****Signature**button at the top right.
4. Type the text of your signature. If you have an HTML signature, click **HTML** at the bottom right and paste the HTML code.
5. Click **Save**. Now you can set this signature as a default one for any of your accounts.

## Switch between multiple signatures

[Mac](#) [Windows](#) [Android](#) [iOS](#)

When you compose an email, your default signature displays at the bottom. If you have multiple signatures, you can select a different one.

Click on the arrow buttons to choose a different signature. Select **Hide****Signature** if you don’t want to use a signature in this email.

## Set default signatures for different accounts

You can choose a different default signature for each of your email accounts. It’s also possible to show no signature by default.

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/general/how-to-find-spark-settings).
2. Select **Composer >****Signatures**.
3. In the **Default Signature** section, select the signature you wish for each account, or choose**Most Recent** or **No Signature**.

## Delete a signature

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/general/how-to-find-spark-settings).
2. Select **Composer >****Signatures**.
3. Click the signature you want to delete and click on the **Delete** button.

1. [Open Spark Settings](https://sparkmailapp.com/general/how-to-find-spark-settings).
2. Select **Composer >****Signatures**.
3. Click the signature you want to delete and click on the **Delete** button.

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588249202708-1588249202708.png)  at the top left.
2. Tap **Settings**![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588249211538-1588249211538.png)  at the bottom.
3. Select **Signatures**.
4. Long tap the signature you want to delete.
5. Tap **Yes** in a pop-up warning message.
6. If you don’t want to use pre-arranged signatures in Spark, disable the **Use Email Signatures** toggle. Now, the signatures won’t display in the email composer.

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588249192346-1588249192346.png)  at the top left.
2. Open **Settings > Signatures**.
3. Select **Edit** at the top right.
4. Tap the minus icon near the signature you want to delete.
5. Choose **Delete**on the right.
6. If you don’t want to use pre-arranged signatures in Spark, disable the **Use Email Signatures** toggle. Now, the signatures won’t display in the email composer.
