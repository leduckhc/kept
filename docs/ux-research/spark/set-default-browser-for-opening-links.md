# Set default browser for opening links

Source: https://sparkmailapp.com/help/general/set-default-browser-for-opening-links
Scraped: 2026-05-31T10:49:59Z

---

You can set the default browser for opening links. When it comes to the default browser, Spark follows the system preferences of your device. For example, you can set Google Chrome as your default browser, and Spark will use it to open links.

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. Open the **System Settings**.
2. Click **Desktop & Dock**.
3. Under **Default web browser**, pick the desired app.

1. Open the **Settings app**.
2. **Apps** **>** **Default Apps**.
3. Search or select the needed browser > **Make "X" your default browser**.

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588252387325-1588252387325.png)  at the top left.
2. Tap **Settings** ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588252381040-1588252381040.png)  at the bottom.
3. Open **Email viewer**.
4. **Open links in** > Select the needed option.

**Note:**If you selected Default, this option will correspond with your **Default apps** selection from the Settings app.

1. Tap the menu icon ![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/6617/direct/1588259201943-1588259201942.png)  at the top left.
2. Open **Settings** **>** **Email Viewer**.
3. **Default Browser** > Select the needed option.

**Note:**If you selected Default, this option will correspond with [your selection from the iOS Settings app.](https://support.apple.com/en-gb/104975)
