# Customize Undo Send Timer

**Source:** https://sparkmailapp.com/help/sending-emails/customize-undo-send-timer

---

You can cancel sending emails within 5 seconds by default when sending emails in Spark. You can customize the Undo Send Timer to change the default time.

[Mac](#) [Windows](#) [Android](#) [iOS](#)

1. [Open Spark Settings](https://sparkmailapp.com/help/general/how-to-find-spark-settings) > **General**.
2. Undo Send Timer > Select the preferred option.

![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/11007/direct/1757434601894-Screenshot%202025-09-08%20at%2010.23.58%E2%80%AFPM.png)

The sent email will remain in your Outbox for the selected time. Click **Undo** to cancel sending the message.
