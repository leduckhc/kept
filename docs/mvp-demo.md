# Kept MVP Demo

## Hero query

> Show important emails I haven’t replied to that mention invoices, contracts, or next week.

## Demo steps

1. Launch Kept desktop app.
2. Connect Gmail.
3. Index recent mail locally.
4. Search across local index.
5. Show ranked thread results with source account.
6. Open one result and show a short local/BYO-AI summary.
7. Open prompt audit and show what was sent or confirm nothing left the device.

## Required proof

- Real Gmail data after connector spike, not mocked mail.
- Search latency shown in terminal or UI.
- Local DB path shown.
- Network boundary explained: Kept server does not receive mail body.
- AI provider shown as user-configured and optional.

## First beta success metric

100 beta users, 20 paid conversions at $5/month or $49/year, with churn reasons captured.
