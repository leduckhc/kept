# How to Triage Your Inbox

Reach inbox zero with Kept's keyboard-driven workflow.

## The 60-second triage

Once you learn the shortcuts, you can process 50+ emails in under a minute:

### The loop

1. Press `j` to move to the next thread
2. Glance at sender + subject + preview
3. Make a decision:
   - `e` — **Archive** (done, no action needed)
   - `s` — **Star** (important, handle later)
   - `#` — **Trash** (junk)
   - `Enter` → read → `r` reply → `e` archive
   - Snooze (via menu) — not ready to deal with now
4. Repeat

### Bulk triage

When you have a batch of similar emails (newsletters, notifications):

1. Click the avatar on the first thread (enters bulk mode)
2. Click more avatars to add to selection
3. Press `e` to archive all, or `#` to trash all
4. Done — 20 emails handled in 2 seconds

## The Triage view (card mode)

For a more focused approach, switch to the **Triage** view:

1. Click the Triage icon in the sidebar
2. Emails appear one at a time as full cards (Superhuman-style)
3. For each card, you see the full message and action buttons:
   - Archive
   - Star
   - Snooze
   - Reply
   - Trash
4. Acting on a card automatically advances to the next one
5. When you've handled everything, you see "Inbox Zero 🎉"

## Pro tips

- **Don't read everything** — the preview snippet in the thread list tells you 90% of what you need to know
- **Use star as "reply later"** — star it, move on, come back to Starred view when you have time
- **Set Aside for reference** — if something is useful but not actionable, press `gb` after setting aside
- **Follow-up reminders** — when you reply and need to track the response, set a "remind if no reply" before archiving
- **Auto Labels first** — set up rules for recurring senders so they're pre-categorized when they arrive

## Sample workflow (morning routine)

```
1. Open Kept (Cmd+Space → "Kept")
2. Auto-sync fires — new emails appear
3. Scan subject lines with j/k
4. Batch-archive all newsletters (bulk select → e)
5. Star 3-4 emails that need replies
6. Trash obvious junk
7. Switch to Starred (gs) and write replies
8. Set follow-up reminders on important outbound
9. Done — inbox is zero, starred is empty
```

Total time: 5-10 minutes for an overnight accumulation of 50-100 emails.
